## üìù How to Update Your App Name

If you choose a new name, update these files:

### 1. **app.json**
```json
{
  "expo": {
    "name": "Freezely",
    "slug": "freezely",
    "ios": {
      "bundleIdentifier": "com.freezely.app"
    }
  }
}
```

### 2. **package.json**
```json
{
  "name": "freezely",
  "description": "Track your freezer inventory"
}
```

### 3. Update all App Store submission files:
- APP_DESCRIPTION.md
- PRIVACY_POLICY.html (title and references)
- Email templates (if created)
- Screenshots (if name appears in them)

### 4. **Assets** (if name appears)
- Regenerate splash screen if it contains app name
- Update any marketing materials
